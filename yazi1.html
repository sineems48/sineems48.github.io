GitHub Pages ile Site Yayınlama

GitHub Pages, GitHub üzerindeki repository’ni kullanarak ücretsiz bir web sitesi yayınlamanı sağlar. Özellikle kişisel bloglar, portfolyolar ve küçük projeler için idealdir.

Adımlar

  1. GitHub’da yeni bir repository oluştur.
  2. Repository içine index.html dosyası ekle.
  3. Settings → Pages sekmesinden GitHub Pages’i aktif et.
  4. Artık siten https://kullaniciadi.github.io adresinde yayında!
